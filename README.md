
# Digital Image Processing – Lab 3

## Topic
Image Noise and Filtering Techniques

## Objective
This lab focuses on understanding different types of image noise and applying filtering techniques to remove noise and enhance image quality.

## Tools Used
- Python
- OpenCV
- NumPy
- Matplotlib
- Google Colab

## Techniques Implemented
- Adding Gaussian Noise
- Mean (Average) Filter
- Gaussian Filter
- Median Filter
- Sharpening Filter

## Learning Outcome
This lab helped in understanding how noise affects digital images and how different filtering techniques can improve image quality while preserving important details.
